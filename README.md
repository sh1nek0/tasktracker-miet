# TaskTracker - Система управления задачами

Простое приложение для управления задачами с графическим интерфейсом, написанное на Python.

## Возможности

- Создание, редактирование и удаление задач
- Установка приоритетов (низкий, средний, высокий)
- Отслеживание статуса задач (Запланирована, В работе, Выполнена)
- Установка сроков выполнения
- Поиск и фильтрация задач
- Отслеживание просроченных задач
- Push-уведомления о просроченных задачах
- Система напоминаний
- Экспорт данных в JSON

## Установка и запуск

### Требования
- Python 3.7+
- tkinter (обычно входит в стандартную поставку Python)

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск приложения

```bash
python run.py
```


### Запуск тестов
```bash
python -m unittest tests.test_task_service -v
```

## Использование

### Создание задачи
1. Нажмите кнопку "Новая задача"
2. Заполните:
   - Название задачи (обязательно)
   - Описание (необязательно)
   - Приоритет (низкий, средний, высокий)
   - Срок выполнения в формате ГГГГ-ММ-ДД (обязательно)
3. Нажмите "Сохранить"

### Управление задачами
- **Редактировать**: Выберите задачу и нажмите "Редактировать" для изменения названия, описания, приоритета, статуса и срока
- **Выполнена**: Выберите задачу и нажмите "Выполнена" для быстрого изменения статуса
- **Удалить**: Выберите задачу и нажмите "Удалить"
- **Просроченные**: Нажмите "Просроченные" для просмотра и управления просроченными задачами
- **Обновить**: Нажмите "Обновить" для обновления списка

### Статусы задач
- **Запланирована** - задача создана, но не начата
- **В работе** - задача выполняется
- **Выполнена** - задача завершена
- **Просрочена** - срок выполнения истек

### Приоритеты
- **Высокий** - срочные и важные задачи
- **Средний** - обычные задачи
- **Низкий** - задачи, которые можно отложить

### Система уведомлений
- **Автоматические уведомления**: Приложение автоматически проверяет просроченные задачи каждый час
- **Ручная проверка**: Нажмите кнопку "Просроченные" для просмотра всех просроченных задач
- **Управление просроченными задачами**: В окне уведомлений можно:
  - Отметить задачу как выполненную
  - Изменить срок выполнения
  - Просмотреть детали просрочки (количество дней)
- **Цветовое выделение**: Просроченные задачи выделяются в основном списке

## База данных

Приложение использует SQLite для хранения данных. База данных создается автоматически при первом запуске в папке `data/`.

### Структура таблицы tasks:
- `id` - уникальный идентификатор
- `title` - название задачи (до 500 символов)
- `description` - описание (до 2000 символов)
- `priority` - приоритет (низкий, средний, высокий)
- `status` - статус (Запланирована, В работе, Выполнена)
- `created_date` - дата создания
- `due_date` - срок выполнения
- `completed_date` - дата выполнения
- `reminder_date` - дата напоминания
- `reminder_sent` - отправлено ли напоминание
